## Baseline (what's strong)

- All pages have unique title (within length, primary keyword first)
- All pages have unique meta description (140-160 chars)
- All pages have canonical URL (note: uses `.html` suffix, kept for consistency)
- All pages have basic OG: og:title, og:description, og:type, og:url
- index.html has a flat LocalBusiness JSON-LD
- sitemap.xml exists at root (lastmod 2026-04-23)
- robots.txt exists at root (Disallow /thank-you.html, Sitemap line)
- _headers file in place (security headers shipped 3 days ago)
- Tailwind via CDN, GSAP for animations - clean and consistent

## Gaps (universal, all 15 pages)

| Gap | Severity | Notes |
|-----|----------|-------|
| `og:image` + width/height/alt missing | HIGH | Social shares render bare. |
| `og:site_name` + `og:locale` missing | MEDIUM | OG completeness. |
| Twitter Card set missing entirely | HIGH | No twitter:card / image. |
| `geo.region` / `geo.placename` / ICBM missing | MEDIUM | Local-SEO signal. |
| `BreadcrumbList` JSON-LD missing on all pages | MEDIUM | Helps SERP breadcrumbs. |
| Skip-to-content link missing | MEDIUM | Accessibility. |
| `<main id="main">` wrapper missing on every page | MEDIUM | Sections sit directly under body. Needs structural wrap. |
| In-body breadcrumb nav above hero (inner pages) | LOW | Reinforces BreadcrumbList. |

## Anti-patterns

- No `aggregateRating` anywhere - good (no verified single-platform review count >= 10).
- No fabricated stats noted in body copy on spot-check.
- No em dashes detected.
- Note: copy uses real, verifiable stats ("2 leads in 8 days", "$45 CPL", "$3,600 first job") - voice already disciplined.
